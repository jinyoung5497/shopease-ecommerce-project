- my-monorepo: 모노레포의 루트 디렉토리, 전체 설정 파일 포함
- apps : 개별 프로젝트 저장소
  - shopease: Fashion e-commerce 프로젝트
    - app: 전역 설정 및 초기화 파일
    - features: 기능별 모듈(api, hooks 포함)
    - pages: 페이지별 컴포넌트
    - shared: 공용 컴포넌트 및 훅
    - store: Zustand 기반 전역 상태 관리
- packages: 프로젝트 공용 패키지
  - ui: 디자인 시스템
    - components: 디자인 시스템 컴포넌트
    - hooks: 시스템 전용 커스텀 훅
